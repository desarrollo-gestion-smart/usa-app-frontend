# All Benefits Group

Base mobile app in Flutter for Android and iOS.

## Stack

- Flutter 3.41
- Dart 3.11
- Material 3
- Google Fonts

## Run

```bash
flutter pub get
flutter run
```

## Current foundation

- App shell with bottom navigation
- Custom visual theme and typography
- Premium-style home screen with reusable cards
- Example sections for dashboard, benefits, and profile
- Widget test and static analysis passing
