# Sophisticated UI Effects

Use this skill when the user wants the interface to feel:
- sophisticated
- luxurious
- futuristic
- immersive
- premium
- highly polished

This skill focuses on effects, depth, texture, and motion language.

## Core Rule

Effects must elevate the product, not distract from it.

The interface should feel expensive, not overloaded.

## Approved Effect Families

### 1. Glassmorphism
Use for:
- floating panels
- premium toolbars
- bottom navigation containers
- contextual overlays

Guidelines:
- use moderate blur
- pair with translucent fills
- add subtle borders
- maintain strong text contrast
- avoid stacking too many glass layers in one viewport

### 2. Spotlight Highlighting
Use for:
- guided tutorials
- onboarding tours
- premium focus states
- drawing attention to one featured element

Guidelines:
- use a dimmed backdrop
- highlight one element clearly
- use warm or elegant accent glow
- do not hide the highlighted target under the tooltip

### 3. Layered Depth
Use for:
- featured cards
- call-to-action blocks
- premium stat modules
- mascot or assistant presentation

Guidelines:
- prefer soft multi-layer shadows
- combine subtle elevation with border definition
- vary depth by importance
- avoid black-heavy muddy shadows

### 4. Luxury Gradients
Use for:
- hero areas
- premium banners
- signature buttons
- highlighted backgrounds

Guidelines:
- keep gradients controlled
- use 2-3 tones max
- prefer deep neutrals, warm golds, sophisticated creams, muted greens, restrained blues
- avoid rainbow or overly saturated gradients

### 5. Microinteractions
Use for:
- hover states
- pressed states
- onboarding transitions
- page switches
- card emphasis

Guidelines:
- animation should be short and smooth
- prefer easeOut and easeInOut curves
- motion should feel responsive, not theatrical
- never make critical UX slower

## Effect Recipes

### Premium Card
- soft background
- 1 subtle border
- 1 ambient shadow
- 1 stronger directional shadow
- optional gradient accent area

### Floating Navigation
- blur backdrop
- rounded container
- soft border
- low-opacity fill
- selected state with subtle emphasis

### Guided Assistant Overlay
- dim screen with controlled opacity
- one highlighted target
- compact modal or tooltip
- supporting mascot image
- minimal copy

### High-End CTA
- elegant fill or dark luxury background
- gentle shadow
- strong contrast text
- refined padding and corner radius
- clear pressed state

## Motion Rules

Prefer:
- 180ms to 320ms for most UI transitions
- smooth opacity transitions
- slight scale or slide transitions
- choreography that follows user intent

Avoid:
- bouncing everywhere
- long dramatic animations
- too many simultaneous animated properties
- flashy movement that breaks premium perception

## Frontend Implementation Guidance

When implementing effects:
- start with visual hierarchy first
- add depth second
- add motion last
- test readability after every effect layer
- keep performance in mind on mobile

## Mobile Safety Checklist

Before finalizing:
- confirm text remains readable over blur or gradients
- confirm highlights do not hide important controls
- confirm shadows are visible on light and dark backgrounds
- confirm animation does not feel slow
- confirm effect layers do not create visual clutter
