# Premium Frontend Polish

Use this skill when a screen already works, but still feels:
- generic
- noisy
- uneven
- visually inconsistent
- almost good, but not premium

This skill is for refinement and critique, not initial layout invention.

## Purpose

Review a frontend screen and improve:
- consistency
- spacing rhythm
- typography balance
- component elegance
- CTA clarity
- premium finish

## Review Mindset

Ask:
1. What is the main action?
2. Does the first viewport communicate it clearly?
3. Which elements feel too loud?
4. Which areas feel empty in a bad way versus intentionally minimal?
5. Do the effects feel refined or just decorative?
6. Does the screen feel like one product or a collection of unrelated widgets?

## What To Polish

### 1. Spacing Rhythm
- align section spacing
- normalize card padding
- reduce awkward gaps
- make vertical rhythm intentional

### 2. Typography
- ensure clear title/subtitle/body hierarchy
- reduce oversized headlines when they overpower the screen
- shorten verbose supporting text
- make premium screens feel editorial, not crowded

### 3. Card Quality
- ensure cards have a clear purpose
- remove duplicate visual treatments
- normalize corner radius language
- keep borders and shadows consistent

### 4. CTA Strategy
- one primary CTA should dominate
- secondary actions should feel secondary
- avoid too many competing button styles

### 5. Color Discipline
- reduce unnecessary accent usage
- keep the palette narrow and intentional
- make sure highlights guide attention, not scatter it

### 6. Effect Restraint
- reduce effects if they weaken clarity
- keep blur, glow, and shadow under control
- preserve one standout premium moment per section

## Premium Quality Checklist

A polished screen should feel:
- calm
- confident
- clear
- intentional
- balanced
- expensive without being flashy

## Anti-Patterns

Watch for:
- too many card styles on one screen
- too much copy above the fold
- oversized widgets covering important content
- highlight overlays hiding the target
- too many bold sections competing at once
- "premium" effects that reduce usability

## Tutorial-Specific Review Rules

When reviewing guided tutorials:
- the highlighted target must remain visible
- tooltip/modal must adapt to available space
- assistant image must support, not dominate
- step copy must be brief and useful
- controls must be obvious and compact

## Output Format

When using this skill, provide:
1. what feels off
2. what should be removed
3. what should be simplified
4. what should be emphasized
5. exact refinement suggestions
